# Projeto de TCC
**Tema:** Arquitetura Resiliente e Escalável para Aplicações na AWS

**Resumo:** 
Este trabalho tem como objetivo construir uma arquitetura resiliente e escalável na plataforma de computação em nuvem AWS (Amazon Web Services) para suportar aplicações que necessitam de alta disponibilidade e desempenho. A arquitetura desenvolvida utiliza instâncias EC2 em conjunto com o Autoscaling, um Application Load Balancer para distribuir o tráfego entre as instâncias, e o monitoramento automatizado com o Amazon CloudWatch. Além disso, são realizadas simulações de carga e falhas, utilizando a ferramenta WRK, para testar a eficiência da solução em condições de estresse.

## Problema
A construção de aplicações web que atendam aos requisitos de alta disponibilidade, escalabilidade e desempenho é um desafio complexo. A escolha de tecnologias e a implementação de uma arquitetura adequada são cruciais para garantir a qualidade do serviço.
Neste cenário, surgem desafios como garantir o dimensionamento adequado da aplicação, de modo a atender à demanda sem desperdício de recursos. A resiliência é essencial para que a aplicação funcione mesmo em caso de falhas. A escalabilidade precisa permitir ajustes na capacidade conforme a demanda varia, enquanto o monitoramento em tempo real é necessário para identificar e resolver problemas rapidamente.
